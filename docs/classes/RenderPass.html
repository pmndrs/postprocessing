setSize

(
  • width
  • height
)

Inherited from Pass: src\passes\pass.js:79

Updates this pass with the main render target's size. This is an abstract method that may be overriden in case you want to be informed about the main render size. The effect composer calls this method when the pass is added and when the effect composer is reset.

Parameters:

  • width Number
    • The width.
  • height Number
    • The height.

Example:

this.myRenderTarget.width = width / 2;
